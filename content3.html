Arduino IoT Project

Proje Açıklaması

Bu proje, Arduino kullanarak nesnelerin internetine bağlanmayı ve uzaktan veri izlemeyi amaçlamaktadır. Kullanıcılar, bu proje ile IoT uygulamalarının temelini öğrenebilir.

Gerekli Malzemeler:

  • Arduino Uno
  • ESP8266 WiFi Modülü
  • Bağlantı kabloları
  • Breadboard (Devre tahtası)

Kod Örneği


// Gerekli kütüphaneleri ekle
#include 

// WiFi bilgileri
const char* ssid = "YOUR_SSID";
const char* password = "YOUR_PASSWORD";

// setup() fonksiyonu
void setup() {
  Serial.begin(115200);
  WiFi.begin(ssid, password);

  while (WiFi.status() != WL_CONNECTED) {
    delay(1000);
    Serial.println("Bağlanıyor...");
  }
  Serial.println("Bağlantı başarılı!");
}

// loop() fonksiyonu
void loop() {
  // Burada sensör verilerini okuyup işleyebilirsiniz
  delay(1000);
}
